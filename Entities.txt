User Score Calculation:

1. Approach Score: (Answer Quality(Provided by AI)*Difficulty weight)/(10*Difficulty Weight)
approachScore = approachScore + 

2. Edge Cases Score: (Number of edge cases mentioned/Total number of edge cases) * (Difficulty Weight/Max Difficulty) * 100
Average of all these scores is the users edge cases score.

3. Consistency Score: 0.4 * active_day_ratio + 0.3 * streak_score + 0.1 * recency_score
	a. active_day_ratio = today - account creation date >= 30 ? (active_days_last_30  / 30) : (active days since 	account creation/(today-account creation date))
	b.streak_score = log(1 + current_streak) / log(31)
	c. recency_score = e^(-days_since_last_active / 14)
